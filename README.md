Jymfony Example
---------------
Basic Jymfony Example

Usage
-----
Install libraries:
```sh
$ yarn
```

Start the web server:
```sh
$ bin/console http:listen localhost 8000
```

Open in your favourite browser this url: http://localhost:8000
